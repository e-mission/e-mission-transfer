# e-mission-transfer
Scripts to import/export data from e-mission instances

1. By sharing an encrypted file on a website
  1. To export:
     1. Export the data in the container
     1. Zip it up
     1. Copy it out
     1. Encrypt it
     1. Copy to a website
 2. To import:
     1. Download the file
     1. Decrypt
